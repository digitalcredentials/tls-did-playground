# tls-did-playground

A playground to provide an example and test the tls-did libraries

## Prerequisite

- Minimum node.js: v14.3.0
- tls-did-registry repository on the same path as tls-did-playground

## Installation

```
npm i
```

## Pre Run

```
npm run ganache
npm run truffle
```

## Run

```
npm run start
```
